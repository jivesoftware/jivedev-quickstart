Jive Dev QuickStart App

The Jive Dev QuickStart App provides new and old Jive Developers with an easy and effortless way to generate add-on packages for their Jive Communities.

A Jive App is an integration that displays a contextually aware UI to expand the functionality of the Jive platform. For more details about this feature, refer to Jive Apps (OpenSocial) in the Jive Community. Custom View Tile

Custom View Tiles offer a dashboard-like experience to augment your Jive place(s). They have contextual data passed in of the person and place they are being viewed by/in. In this out of the box example, we have prewritten code for getting the URL of the Tile for absolute links, context of the viewer of the Tile at the current time, context of the place that's being viewed, and performing OSAPI HTTP requests.

For more details about Custom View Tiles and extending beyond the output of this builder, read Creating Custom View Tiles.

Simple Stream Integration

Simple Stream Integrations offer an easy way to generate activities into Jive to offer the full collaborative power of Jive by bringing in activity notifications from other systems. They require the external service to have an outbound webhook to be available and require only minimal JavaScript to translate the message from the external system to the content needed by Jive to create an activity.

For more details, take a look at Dev Tool: Using the Simple Stream Intrgation Builder with StatusPage.io series.

OAuth Token Generator

In order to make admin level REST API calls, your users will need to have an access token. This add-on allows you to create a bare add-on that can provide you with a client ID and client secret to get to an OAuth 2 Access and Refresh Tokens.

For more information on completing the OAuth grant cycle with Jive, check out OAuth 2.0-Creating a Client and Managing Access
